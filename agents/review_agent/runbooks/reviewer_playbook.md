# Reviewer Playbook

## Goals
- preserve candidate quality and client trust
- prevent unsupported submissions
- improve model calibration through high-quality feedback

## Reviewer Checklist
1. Validate evidence provenance.
2. Confirm must-have skill coverage.
3. Check for data freshness issues.
4. Approve, reject, or request recruiter clarification.
5. Tag the root cause for downstream model analysis.
