# Review Queue Definitions

## `entity_resolution`
- ambiguous engineer identity merges
- conflicting company normalization results
- stale or duplicate repository linkages

## `low_confidence_shortlist`
- top candidates lack enough verified evidence
- opening requirements were inferred from weak intake notes
- talent pool is unusually small for the requested search

## `outbound_approval`
- client submissions awaiting recruiter signoff
- outreach drafts referencing sensitive or uncertain data
