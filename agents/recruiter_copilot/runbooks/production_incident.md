# Recruiter Copilot Incident Runbook

## Common Incidents
- retrieval latency spike
- stale ATS sync
- CRM writeback failures
- explanation generation missing evidence

## Immediate Response
1. Disable auto-draft delivery.
2. Keep search enabled in read-only mode if evidence remains trustworthy.
3. Notify talent operations and staffing leadership.
4. Route urgent roles to manual recruiter handling.
