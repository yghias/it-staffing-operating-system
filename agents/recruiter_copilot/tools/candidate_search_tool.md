# Candidate Search Tool

## Purpose
Retrieve a ranked candidate pool from structured filters, graph traversals, and vector similarity.

## Inputs
- normalized role title
- required skills
- required technologies
- location constraints
- availability window
- compensation band

## Outputs
- candidate identifiers
- relevance score
- evidence bundle
- similarity features
- prior placement context

## Failure Modes
- source freshness below SLA
- insufficient candidate pool size
- confidence below search threshold
