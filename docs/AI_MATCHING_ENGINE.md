# AI Matching Engine

## Matching Stages

1. Hard filters for location, availability, authorization, rate band, and must-have requirements.
2. Semantic retrieval over resumes, projects, recruiter notes, and job descriptions.
3. Graph-aware reranking using skill coverage, technology depth, recency, related placements, and domain adjacency.
4. Explanation generation for recruiter review.

## Scoring Signals
- Skill overlap and proficiency
- Technology recency
- Domain relevance
- Similar project history
- Prior placement outcomes
- Recruiter feedback

## Human Controls
- Confidence threshold routing
- Mandatory recruiter approval before candidate submission
- Feedback capture for accepted and rejected recommendations
