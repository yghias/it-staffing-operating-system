# Monitoring and Alerting

## Core alarms

- ATS sync freshness exceeds 90 minutes
- CRM sync freshness exceeds 60 minutes
- warehouse build failure
- match-score publication failure
- review queue SLA breach
- API error rate above threshold

## Response model

- source freshness alerts page platform operations
- warehouse and publication failures notify data platform and staffing operations
- review queue SLA breaches notify desk operations leads
